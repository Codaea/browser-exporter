Metrics URL

The URL that the extension pushes to. This is UNSET by default for privacy reasons. the public instance is https://browser.codaea.com To selfhost, check out the documentation here. This value is synced across all devices with the extension installed.

Your Identifier

On install, a unique identifier is created for your device. it is entirely random, and has no way to be tracked back to you besides you knowing it. if you would like request a new one, you can do so here.
